# Compromiso Mi 0km - Grupo Sur Automotores

Landing page comercial para captar leads de Volkswagen, Peugeot y Fiat.

## Comandos

Instalar dependencias:

```bash
npm install
```

Probar localmente:

```bash
npm run dev
```

Crear versión final:

```bash
npm run build
```

## Publicación en Vercel

1. Subir esta carpeta a un repositorio de GitHub.
2. Entrar a Vercel.
3. Add New Project.
4. Importar el repositorio.
5. Deploy.
6. En Settings > Domains, agregar:
   - compromisomi0km.com.ar
   - www.compromisomi0km.com.ar
